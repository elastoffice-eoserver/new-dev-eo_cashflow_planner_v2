üìä Overview

eoCashflow Planner V2 is a comprehensive cashflow management solution that significantly improves upon the original eo_cash_reports module. Plan your income and expenses, track budgets, forecast future cashflow, and export professional Excel reports.

‚ú® Key Features

1. Planned Items

  • Plan income and payment items with due dates
  • Set priorities (Low/Medium/High)
  • Categorize with hierarchical categories
  • Link to partners and invoices
  • Track states (Planned/Paid/Cancelled)

2. Recurring Items

  • Auto-generate recurring cashflow items (rent, salaries, subscriptions)
  • Recurrence patterns: Daily, Weekly, Monthly, Quarterly, Yearly
  • Custom intervals (every 2 months, every 6 months, etc.)
  • One-click generation button

3. Budget Tracking

  • Create budgets by category and time period
  • Automatic usage calculation from planned items
  • Real-time variance analysis
  • Color-coded warnings (green/orange/red)

4. Hierarchical Categories

  • Parent/child category structure
  • 17 pre-loaded default categories
  • Separate trees for Income and Payment
  • Easy navigation and filtering

5. Advanced Reports

  • Cashflow Overview: Combines planned items + customer/supplier invoices
  • Cashflow Forecast: Project 3/6/12 months with running balance
  • Budget Analysis: Compare planned vs actual with variance
  • Advanced filtering, grouping, and detail views
  • NEW: Excel Export with professional formatting

6. Invoice Integration

  • Combine planned items with actual invoices in reports
  • Toggle data sources (planned items on/off, invoices on/off)
  • Shows document numbers and residual amounts
  • Jump to invoice/planned item with one click

üÜö Improvements Over V1 (eo_cash_reports)

  • Recurring Items: Automatically generate monthly/quarterly/yearly transactions
  • Budget Tracking: Track planned vs actual spending with variance analysis
  • Enhanced Forecasting: Project future cashflow with running balance
  • Better Categories: Hierarchical structure with parent/child relationships
  • Excel Exports: Professional XLS reports with color coding
  • Better Code Quality: Built using EOServer Pattern Library
  • Enhanced Security: Three user groups (User/Manager/Director)

üì¶ Installation

Prerequisites:

  • Odoo 7.0 server
  • Python 2.7 environment
  • report_xls module (for Excel exports)

Install using Makefile:

make oeupdate db=YOUR_DATABASE mod=eo_cashflow_planner_v2

Update after changes:

make oeupdate db=YOUR_DATABASE mod=eo_cashflow_planner_v2
make restart

üöÄ Quick Start

  1. Assign user groups: Settings ‚Üí Users ‚Üí Application ‚Üí Cashflow Planning
  2. Review categories: Accounting ‚Üí Cashflow Planning ‚Üí Configuration ‚Üí Categories
  3. Create planned items: Accounting ‚Üí Cashflow Planning ‚Üí Planned Items ‚Üí Create
  4. Run reports: Accounting ‚Üí Cashflow Planning ‚Üí Reports
  5. Export to Excel: Click "Export to XLS" button on any report

üìä Report Examples

Cashflow Overview Report

Combines planned items with customer and supplier invoices:

  • Filter by date range, type, state, category, partner
  • Toggle data sources (planned items, invoices)
  • Color-coded: Blue = Income, Red = Payment
  • Shows totals: Total Income, Total Payment, Net Cashflow
  • Detail button for advanced filtering (550 line limit)
  • Export to Excel with professional formatting

Budget Analysis Report

Compare budget vs actual spending:

  • Filter by fiscal year, date range, category, variance
  • Color-coded tree view:
    • Green: < 50% used (good)
    • Orange: >= 90% used (warning)
    • Red: > 100% used (over budget)
  • Variance amounts and percentages
  • Export to Excel with color-coded variance analysis

Cashflow Forecast Report

Project future cashflow:

  • Forecast for 3, 6, or 12 months
  • Running balance calculation
  • Include/exclude recurring items
  • Monthly/weekly/quarterly breakdown

üîê Security

Three user groups with role-based access:

  • User: View and create planned items, run reports
  • Manager: Full access to budgets, recurring items, categories
  • Director: Strategic oversight, full system access
